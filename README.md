# ☕ Coffee Shop Sales Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data (50,000+ orders) using **Power BI** to uncover customer behavior, sales trends, and business insights.

The dashboard provides a clear and interactive view of key performance metrics to support data-driven decision-making.

---

## 📊 Key Metrics
- **Total Sales:** 122K+
- **Total Orders:** 4K
- **Average Order Value (AOV):** 31.38

---

## 📈 Dashboard Features

### 🔹 Sales Analysis
- Monthly sales trends
- Identification of peak sales periods (March & October)

### 🔹 Customer Segmentation
- High Buyers
- Mid Buyers
- Low Buyers  
➡️ High Buyers contribute the majority of revenue (~85K)

### 🔹 Payment Insights
- Card payments dominate (95%+)
- Minimal cash transactions

### 🔹 Product Performance
- Top 5 selling products:
  - Latte
  - Americano with Milk
  - Cappuccino
  - Americano
  - Hot Chocolate
